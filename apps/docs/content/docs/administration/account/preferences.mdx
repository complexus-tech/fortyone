## User Preferences

Personalize your FortyOne environment to suit your workflow and visual comfort. Changes made here typically only affect your own user account.

### Appearance

- **Theme**: Choose between light mode, dark mode, or an option to sync with your operating system's theme settings.
  - _Dark Mode_: Reduces eye strain in low-light environments.
  - _Light Mode_: Offers a traditional bright interface.
  - _System Sync_: Automatically matches your OS theme.

### Automations

Configure how tasks are automatically handled based on certain actions. These settings can help streamline your workflow.

| Automation                                          | Description                                                                                                 | Control       |
| :-------------------------------------------------- | :---------------------------------------------------------------------------------------------------------- | :------------ |
| **Auto-assign to self**                             | When creating new tasks, automatically assign them to yourself by default.                                  | Toggle On/Off |
| **On git branch copy, move task to started status** | After copying the git branch name for a task, automatically move the task to the "Started" workflow status. | Toggle On/Off |
| **On git branch copy, assign to yourself**          | After copying the git branch name for a task, automatically assign the task to yourself.                    | Toggle On/Off |
