## Teams Management

To configure a specific team:

1.  Navigate to **Settings > Teams** (under Features).
2.  Click on the name of the team you wish to manage.

This opens the detailed settings for that team, organized into the following tabs:

## Team Configuration Tabs

<Tabs items={['General', 'Members', 'Workflow', 'Danger Zone']}>
  <Tab value="General">
    Manage basic information for the team.

    -   **Team Name**: The display name for the team (e.g., Engineering).
    -   **Team Code**: A short prefix for team items (e.g., ENG).
    -   **Team Color**: Visual identifier for the team.
    -   **Private Team (Toggle)**:
        -   **On**: Team is private, visible only to its members and admins.
        -   **Off**: Team is public, visible to everyone in the workspace.

    Save changes if prompted.

  </Tab>
  <Tab value="Members">
    Manage team membership.

    -   **Team Members List**: View current team members (name, email, role).
    -   **Add Member Button**: Add existing workspace members to this team.
    -   **Manage Individual Members**:
        -   Remove member from the team.

  </Tab>
  <Tab value="Workflow">
    Configure the team-specific workflow for tasks.

    -   **Workflow Categories**: Organized stages (e.g., Backlog, Unstarted, Started, Completed, Paused, Cancelled).
    -   **Workflow States**: Specific statuses within categories (e.g., `To Do`, `In Progress`). Task counts per state may be visible.
    -   **Customization**:
        -   **Add State (`+` icon)**: Create new states within categories.
        -   **Edit/Delete State**: Rename, delete, or set a state as default for its category.

    This workflow applies to all tasks in the team.

  </Tab>
  <Tab value="Danger Zone">
    Critical operations for the team.

    -   **Delete Team**:
        -   Permanently removes the team and its configurations.
        -   Associated items will be deleted
        -   Requires confirmation (e.g., typing team name). Irreversible.

    Use with caution.

  </Tab>
</Tabs>
