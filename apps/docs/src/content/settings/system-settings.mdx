# System Settings

Learn how to configure system-wide settings for your organization.

## Organization Settings

Manage organization-level configurations:

1. Navigate to Settings > Organization
2. Configure company information and branding
3. Set default permissions and access controls
4. Manage license allocation and usage
5. Configure security and authentication settings

## Workflow Configuration

Customize workflows to match your processes:

- **Story types**: Define custom types beyond the defaults
- **Status workflows**: Create and modify status flows for different story types
- **Custom fields**: Add organization-specific data fields to stories
- **Templates**: Create templates for common story types

## Integration Settings

Connect the system with other tools:

1. Navigate to Settings > Integrations
2. View available integrations
3. Configure connection details for each service
4. Set up synchronization options
5. Test connections to ensure proper functionality
