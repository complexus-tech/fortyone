# Basic Search

Learn how to quickly find what you're looking for using the search functionality.

## Quick Search

Access quick search by:

1. Clicking the search icon in the top navigation
2. Using the keyboard shortcut (Ctrl+K or Cmd+K)
3. Typing in the search box from any page

## Search Syntax

Optimize your searches with these techniques:

- Use quotes for exact phrases: `"release planning"`
- Include specific fields: `assignee:john type:bug`
- Use logical operators: `priority:high OR priority:medium`
- Exclude terms: `-resolved`
- Use wildcards: `integrat*` to match integrate, integration, etc.

## Recent Searches

The system saves your recent searches for quick access:

1. Open the search box
2. View recent searches below the search field
3. Click a recent search to run it again
4. Pin important searches to keep them available
