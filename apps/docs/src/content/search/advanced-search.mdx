# Advanced Search

Learn how to use advanced search features to find exactly what you need.

## Advanced Search Filters

Access advanced search by:

1. Opening the search box
2. Clicking "Advanced" to show additional filters
3. Setting specific criteria for each field

## Filter Types

The system provides various filter types:

- **Text fields**: Story title, description, comments
- **Selectors**: Type, status, priority, assignee
- **Date ranges**: Created, updated, due dates
- **Numeric ranges**: Story points, time estimates
- **Boolean filters**: Has attachments, has subtasks

## Saving Searches

Save complex searches for later use:

1. Configure your search parameters
2. Click "Save Search"
3. Name your search and add a description (optional)
4. Choose whether to share with your team
5. Access saved searches from the search dropdown
