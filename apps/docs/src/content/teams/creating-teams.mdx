# Creating and Managing Teams

Learn how to set up effective team structures.

## Creating a Team

1. Navigate to the Teams section
2. Click "Create Team"
3. Set team details:
   - Team name: A descriptive, unique name
   - Description: The team's purpose and focus
   - Team avatar: Visual identification (optional)
   - Team lead: The primary coordinator
4. Click "Create" to save the team

## Adding Members

1. Open the team settings
2. Navigate to the "Members" tab
3. Click "Add Member"
4. Search for users by name or email
5. Select roles for each member
6. Send invitations to join

## Team Settings

Configure team-specific settings:

- Default story types and workflows
- Sprint cadence and duration
- Working days and capacity
- Integration preferences
- Notification settings
