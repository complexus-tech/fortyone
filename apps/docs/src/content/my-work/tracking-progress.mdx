# Tracking Your Progress

Learn how to monitor your work and productivity over time.

## Updating Task Status

Keep your tasks up to date:

1. Open the task you're working on
2. Update the status using the dropdown menu
3. Add comments about your progress
4. Log time spent (if time tracking is enabled)
5. Upload attachments or documentation as needed

## Progress Metrics

The system provides personal metrics to help you understand your work patterns:

- Stories completed over time
- Average time to completion
- Distribution of work by type
- Workload trends

## Personal Dashboard

Customize your personal dashboard with widgets:

1. Navigate to My Work > Dashboard
2. Click "Customize" in the top right
3. Add, remove, or rearrange widgets
4. Configure each widget's settings
5. Save your dashboard layout
