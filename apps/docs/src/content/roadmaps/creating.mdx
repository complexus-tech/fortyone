# Creating Roadmaps

Learn how to build effective roadmaps that communicate your plans and priorities.

## Creating a Roadmap

1. Navigate to the Roadmaps section
2. Click "New Roadmap"
3. Set roadmap details:
   - Title: A descriptive name for the roadmap
   - Time period: The duration covered by the roadmap
   - Teams: Which teams' work to include
   - View settings: Timeline, kanban, or list view
4. Click "Create" to save the roadmap

## Adding Items to the Roadmap

Add different types of items to your roadmap:

- **Epics**: Large bodies of work that may span multiple sprints
- **Features**: Specific capabilities being delivered
- **Milestones**: Key delivery points or events
- **Releases**: Planned software releases

## Organizing the Timeline

1. Drag items onto the timeline
2. Adjust start and end dates by dragging item edges
3. Group related items together
4. Create dependencies between items
5. Add color-coding to represent different categories or statuses
