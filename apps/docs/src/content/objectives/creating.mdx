# Creating Objectives

Learn how to set effective objectives that align with your organization's strategy.

## Creating an Objective

1. Navigate to the Objectives section
2. Click "New Objective"
3. Fill in the objective details:
   - Title: A clear, concise statement of the goal
   - Description: Additional context and purpose
   - Time period: When the objective should be achieved
   - Owner: Person or team responsible
   - Key results: Measurable outcomes (see below)
4. Click "Create" to save the objective

## Setting Key Results

Each objective should have 2-5 key results that:

- Are specific and measurable
- Have a clear target value
- Are challenging but achievable
- Can be objectively evaluated

Example:

```
Objective: Improve application performance
Key Results:
- Reduce page load time from 3s to under 1s
- Decrease API response time by 50%
- Improve lighthouse performance score from 75 to 95
```

## Aligning Objectives

Objectives should align with:

- Company-level strategic goals
- Department or team priorities
- Individual team member growth plans
